JLPT Kanji Dialogue Lab

이 프로젝트는 JLPT N1/N2 한자 어휘를 단순 암기 카드가 아니라 문맥이 있는 대화 흐름으로 학습하게 만드는 웹앱입니다. 학습자는 일본어 대사, 후리가나, 한국어 번역, 단어 연결 정보를 한 화면에서 확인하고 바로 퀴즈로 복습할 수 있습니다.

구현 방식도 실용적입니다. 정적 프론트엔드에 Supabase를 붙이되, 데이터베이스가 준비되지 않았을 때는 로컬 시드 데이터로 동작하도록 설계해 배포 안정성을 챙겼습니다. SQL 스키마, 시드 파일, JSON 원본, 시드 생성 스크립트를 분리해 콘텐츠 유지보수 흐름까지 고려한 구성이 보입니다.

학습 콘텐츠와 정적 배포의 균형을 잡은 프로젝트로, 앞으로 데이터가 늘어나더라도 구조를 유지하며 확장하기 쉬운 형태라는 점이 강점입니다.

SumDoku Fusion Web

SumDoku Fusion Web은 6x6 Sudoku 규칙과 케이지 합 제약을 결합한 퍼즐을 브라우저에서 바로 플레이할 수 있게 만든 정적 웹앱입니다. 별도 백엔드 없이도 퍼즐 입력, 검증, 힌트, 초기화, 정답 보기 흐름이 모두 동작합니다.

핵심은 복합 규칙 퍼즐을 너무 무겁지 않은 인터페이스로 정리했다는 점입니다. 키보드 입력과 숫자 패드, 상태 검사 같은 요소를 통해 실제 플레이 경험에 집중했고, 배포 역시 정적 파일만으로 끝낼 수 있게 유지했습니다.

학습용 도구와는 결이 다른 게임형 프로젝트라서, 계정 전체 포트폴리오 안에서 작업 폭을 보여주는 역할도 합니다.

Developer Tool Audit

Developer Tool Audit는 Windows 환경에서 주요 개발 도구를 자동으로 찾아 설치 상태만이 아니라 실제 실행 경로, 설치 버전, 관련 환경변수, 최신 공개 버전 대비 outdated 여부까지 한 번에 보여주는 GUI 프로그램입니다. Git, VS Code, Python, Java, Node.js, Go, Rust, .NET, Maven, Gradle, Docker 같은 대표 도구를 우선 점검하고, 새로 발견한 개발 명령도 이후 검사 목록에 자동으로 학습하도록 설계되어 있습니다.

이 프로젝트의 강점은 "개발 환경의 가시화"에 있습니다. 터미널에서 명령어를 하나씩 입력하지 않아도, 어떤 도구가 어디에서 실행되고 있는지, 환경변수가 올바른지, 버전이 얼마나 오래되었는지를 표 형태로 바로 확인할 수 있습니다. GUI 실행뿐 아니라 JSON 출력과 exe 패키징까지 고려해 두어서 개인 PC 점검, 실습실 관리, 팀 장비 표준화 같은 용도로 확장하기 좋습니다.

Windows 환경에서 바로 실행할 수 있는 `DevToolAudit.exe`도 함께 제공하고 있어, 소스 코드를 읽는 용도뿐 아니라 실제로 내려받아 실행해 보는 용도까지 고려한 프로젝트입니다. 개발 도구가 여러 개 섞여 있는 PC를 점검하거나, 새로운 작업 환경을 빠르게 확인해야 할 때 특히 유용한 성격의 프로그램입니다.
